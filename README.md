# sp26-team-09 — CS 152 Spring 2026 Reference Repository

This is the **reference / starter repository** for the CS 152 Trust and Safety group project. It shows the directory layout and starter files the teaching staff expects from each team's repo. Students have **read-only access** — copy the structure into your own team repo rather than pushing here.

## Directory layout

```
sp26-team-09/
├── docs/
│   └── PRD.md            — PRD template (copy into your team repo and fill in)
├── src/                   — your application code (framework of your choice)
├── evals/                 — labeled test sets, evaluation scripts, results
├── poster/                — final Milestone 3 poster (PDF + source)
├── CI_CD.md               — step-by-step CI/CD setup: GitHub → Google Cloud Run
└── README.md              — this file
```

Each subdirectory has its own `README.md` explaining what goes inside.

## What to do with this repo

1. Read the milestone specs on the course website:
   - [Milestone 2](https://stanfordcs152.github.io/milestone2/)
   - [Milestone 3](https://stanfordcs152.github.io/milestone3/)
2. Copy `docs/PRD.md` into your own team repository at `docs/PRD.md` and fill in every section.
3. Follow [`CI_CD.md`](./CI_CD.md) to stand up automatic deployments from GitHub to Google Cloud Run (or Vercel / Cloudflare Pages — both also acceptable).
4. Use the `src/`, `evals/`, and `poster/` conventions in your own repo so the teaching staff knows where to find things when grading.
